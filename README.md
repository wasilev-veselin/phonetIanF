# PhonetIanF

Angular 19 app that lists Ice and Fire books with NgRx-managed list, detail, and favorites.

## Quick start
```bash
npm install
npm start      

## Features
- List Page: Displays items from the API with a search bar for quick filtering and favorite toggling.
- Detail Page: Shows detailed information for a selected item.
- Favorites Page: View/manage favorites; users can add/remove items.
- Uses NgRx for state management (books list, detail, favorites).
